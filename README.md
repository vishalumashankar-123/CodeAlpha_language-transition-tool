# Language Translation Tool

## Description
The Language Translation Tool is a simple web application that translates text from one language to another. Users can enter text, choose the source and target languages, and view the translated result instantly using a translation API.

## Features
- Enter text to translate
- Select source language
- Select target language
- Display translated text
- Copy translated text to the clipboard

## Technologies Used
- HTML
- CSS
- JavaScript
- MyMemory Translation API

## How to Run
1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension (if not already installed).
4. Right-click `codtask1.html` and select **Open with Live Server**.
5. Enter text, choose the source and target languages, and click **Translate**.
